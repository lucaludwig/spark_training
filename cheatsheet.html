Spark Architektur

KomponenteRolle
DriverOrchestriert, plant, verteilt Tasks an Executors
ExecutorFuehrt Tasks aus, speichert Daten (JVM auf Worker)
SlotEin Thread/Core in einem Executor
SparkSessionEinstiegspunkt (spark), nur fuer read/sql/create
Hierarchie: Job > Stage > Task
  • Job = ausgeloest durch eine Action
  • Stage = getrennt durch Wide Transformations (Shuffle)
  • Task = eine Stage auf einer Partition (parallelisierbar)

repartition()coalesce()

Joins

Reihenfolge: DataFrame.join(other, condition, how)

Broadcast Join (kleiner DF an alle Executors):

from pyspark.sql.functions import broadcast itemsDf.join(broadcast(smallDf), condition, "inner")

Auto-Broadcast Threshold (Default: 10MB = 10485760 Bytes):

spark.conf.set("spark.sql.autoBroadcastJoinThreshold", 20971520) # 20 MB in BYTES, nicht MB!

Read & Write

# Read df = spark.read.parquet("/path/file.parquet") df = spark.read.json("/path/file.json") df = spark.read.format("csv").option("header","true").load(path) df = spark.read.schema(mySchema).parquet(path) df = spark.read.option("recursiveFileLookup","true").parquet(path) # Write (IMMER ueber .write!) df.write.format("parquet").mode("overwrite").save(path) df.write.mode("append").parquet(path) # FALSCH: df.format("parquet").save(path) -- fehlt .write!

Modi: "overwrite", "append", "ignore", "error" (default)
